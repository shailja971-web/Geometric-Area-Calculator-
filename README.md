# Geometric-Area-Calculator-

#Project Title: Geometric Shape Area Calculator

Project Description:

The Geometric Shape Area Calculator is a Java-based application designed to calculate the area of different geometric shapes such as circles, squares, rectangles, and triangles. The project demonstrates the principles of Object-Oriented Programming (OOP), including classes, objects, inheritance, and polymorphism.

The user provides the required dimensions for a selected shape, and the program computes and displays the corresponding area using the appropriate mathematical formula. Each shape is represented as a separate class with its own area calculation method, making the program modular, reusable, and easy to extend.

This project helps students understand OOP concepts while applying mathematical computations in a practical application. It can be further enhanced by adding more shapes, perimeter calculations, graphical interfaces, and input validation features.
